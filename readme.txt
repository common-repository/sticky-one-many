Sticky One to Many Elements in Your Website

== Description ==

Sticky your element/div any in your website.you can stick one or many elements.
you will find Sticky elements setting option in <strong>Appearance > Sticky Options<strong>

You will find a <a target="_blank" href="https://plugins-demo.nayeemriddhi.info/sticky-one-to-many/">Demo</a> here

<strong>Plugin Features </strong>

* Make one more Sticky elements in your site.
* You can give top spacing of your sticky elements
* You can also give where to be the element stop.
